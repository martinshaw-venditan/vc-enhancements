### How to install.

1. Run 'yarn build'.
2. Go to 'chrome:extensions'.
3. Enable 'Developer Mode' (top right).
4. Click 'Load unpacked'.
5. Select the 'dist' folder and click 'Select'.
6. The extension will work on the relevant VC pages.
